# Hodowla i Walki Kucyków

## Kompilacja
Potrzebne pakiety:
 - sqlite
 - pkg-config
 - g++
 - make

Aby skompilować program wystarczy użyć polecenia
```sh
make
```


Jeżeli masz menedżera pakietów `nix`, to cały proces instalacji pakietów
i kompilacji można wykonać używając
```sh
nix develop
make
```
